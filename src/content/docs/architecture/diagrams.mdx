### Multi-Instance Modules

`tenant_modules` table:

| module_name | instance_name | table_prefix |
|---|---|---|
| invoices | invoices-in | `invoices_in_` |
| invoices | invoices-out | `invoices_out_` |

Resulting tables in `tenant_acme`:

- `invoices_in_invoices`, `invoices_in_invoice_items`
- `invoices_out_invoices`, `invoices_out_invoice_items`

Routes: `GET /api/v1/tenants/{tenant_id}/i/{instance}/invoices`
Perms: `invoices-in.read.all` (derived from `invoices.read.all`)
Events: `invoices-in.created` (derived from `invoices.created`)

|---|---|---|---|

| | POST | `/api/v1/auth/refresh` | Refresh access token |
| | POST | `/api/v1/auth/logout` | Revoke session |
| | GET | `/api/v1/auth/me` | Current user identity |
| | POST | `/api/v1/auth/oauth2/callback` | OAuth2 code exchange |
| | POST | `/api/v1/auth/magic-link/request` | Send magic link |
| | POST | `/api/v1/auth/magic-link/verify` | Verify magic link |
| | POST | `/api/v1/auth/webauthn/register` | Start WebAuthn registration |
| | POST | `/api/v1/auth/webauthn/login` | Start WebAuthn login |
| **Users** | POST | `/api/v1/users` | Register |
| | GET | `/api/v1/users` | List (scoped) |
| | GET | `/api/v1/users/{id}` | Get |
| | PUT | `/api/v1/users/{id}` | Update profile |
| | POST | `/api/v1/users/{id}/change-password` | Change password |
| **Tenants** | POST | `/api/v1/tenants` | Create (+ schema + seed) |
| | GET | `/api/v1/tenants` | List |
| | GET | `/api/v1/tenants/{id}` | Get |
| | PUT | `/api/v1/tenants/{id}` | Update |
| **Modules** | GET | `/api/v1/tenants/{tenant_id}/modules` | List tenant modules |
| | GET | `.../modules/{name}` | Get module status |
| | PUT | `.../modules/{name}/enable` | Enable module |
| | PUT | `.../modules/{name}/disable` | Disable module |
| | PUT | `.../modules/{name}/config` | Update module config |
| | POST | `.../modules/{name}/instances` | Create instance |

| | GET | `/openapi` | OpenAPI 3.1 spec |
